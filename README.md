## Advanced healthcare directive (Patsiendi elulõpu tahteavaldus - PET)

FHIR IG to implement patient's living will in Estonian health portal

Questionnaire of the will is built in QRE IG: https://github.tehik.ee/ig-ee-qre/ 

Deployed FHIR autobuilder build can be viewed here: https://build.fhir.org/ig/TEHIK-EE/ig-ee-pet
